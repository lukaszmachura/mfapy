# pymfa
Python for MultiFractal Analysis

Functions
---------
* mfdfa: original MFDFA function, as in [1] and [2]
* emdmfdfa: EMD-based version of the above, see [5] and [6]
* fmfssc: focus based version, as in [3] and [4]

* dataplot_qRMS
* get_spectrum
